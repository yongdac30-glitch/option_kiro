# 期权风险监控系统 - 使用说明

## 系统概述

期权风险监控系统是一个专业的期权组合风险管理工具，帮助交易者实时监控期权持仓在不同市场条件下的盈亏情况。

## 主要功能模块

### 1. 持仓管理模块

**功能**：管理所有期权合约持仓

**操作步骤**：

1. **添加新持仓**
   - 点击右上角"添加持仓"按钮
   - 填写表单：
     - 标的代码：输入股票代码（如 AAPL、TSLA、SPY）
     - 期权类型：选择 Put 或 Call
     - 行权价：输入期权执行价格
     - 到期日：选择期权到期日期
     - 合约数量：
       - 正数 = 买入（Long）
       - 负数 = 卖出（Short）
       - 例如：-1 表示卖出1张合约
     - 开仓价格：输入支付或收到的权利金
   - 点击"添加持仓"保存

2. **编辑持仓**
   - 在持仓列表中找到要修改的持仓
   - 点击"编辑"按钮
   - 修改信息后保存

3. **删除持仓**
   - 点击"删除"按钮
   - 确认删除操作

**注意事项**：
- 卖出期权（Sell Put/Call）使用负数表示数量
- 每张期权合约代表100股标的资产
- 开仓价格是每股的权利金价格

### 2. 控制面板

**功能**：设置计算参数并触发盈亏分析

**参数说明**：

1. **标的代码**
   - 从已添加的持仓中自动提取
   - 选择要分析的标的资产

2. **当前价格**
   - 输入标的资产的当前市场价格
   - 用于计算期权的理论价值

3. **隐含波动率（IV）**
   - 输入百分比形式（如 25 表示 25%）
   - 反映市场对未来价格波动的预期
   - 典型范围：15% - 50%
   - 高波动率 = 期权价值更高

4. **计算盈亏**
   - 点击按钮触发计算
   - 系统会在当前价格±50%范围内生成100个价格点
   - 计算每个价格点的组合盈亏

### 3. 风险指标卡片

**显示内容**：

1. **当前盈亏**
   - 基于输入的当前价格计算
   - 绿色 = 盈利，红色 = 亏损
   - 显示当前价格

2. **最大潜在亏损**
   - 在价格范围内可能出现的最大亏损
   - 显示触发该亏损的价格点
   - 红色警告显示

3. **最大潜在盈利**
   - 在价格范围内可能获得的最大盈利
   - 显示触发该盈利的价格点
   - 绿色显示

**风险警告**：
- 当最大潜在亏损超过 $10,000 时，系统会显示警告提示

### 4. 盈亏曲线图

**图表元素**：

1. **蓝色曲线**：组合总盈亏随价格变化的曲线
2. **蓝色实线**：当前价格参考线
3. **红色虚线**：最大亏损点标记
4. **灰色虚线**：零盈亏线（盈亏平衡）

**如何解读**：
- X轴：标的资产价格
- Y轴：组合盈亏金额（美元）
- 鼠标悬停可查看具体价格点的盈亏值
- 曲线向上 = 价格上涨时盈利
- 曲线向下 = 价格下跌时亏损

## 使用场景示例

### 场景1：卖出看跌期权（Sell Put）

**策略**：看好某股票，卖出看跌期权收取权利金

**操作**：
1. 添加持仓：
   - 标的：AAPL
   - 类型：PUT
   - 行权价：$150
   - 到期日：2024-03-15
   - 数量：-1（卖出）
   - 开仓价格：$2.50

2. 设置参数：
   - 当前价格：$155
   - 隐含波动率：25%

3. 查看结果：
   - 如果股价保持在$150以上，获得最大盈利 $250（$2.50 × 100）
   - 如果股价跌破$150，开始出现亏损
   - 最大亏损出现在股价大幅下跌时

### 场景2：多腿策略（组合持仓）

**策略**：牛市价差（Bull Put Spread）

**操作**：
1. 卖出较高行权价的Put：
   - 行权价：$150
   - 数量：-1
   - 开仓价格：$3.00

2. 买入较低行权价的Put：
   - 行权价：$145
   - 数量：+1
   - 开仓价格：$1.50

3. 查看组合盈亏：
   - 最大盈利：$150（净权利金收入）
   - 最大亏损：$350（行权价差 - 净权利金）
   - 盈亏平衡点：$148.50

## 计算原理

### Black-Scholes 模型

系统使用 Black-Scholes 公式计算期权理论价格：

**Put 期权价格**：
```
P = K·e^(-rT)·N(-d2) - S·N(-d1)

其中：
d1 = [ln(S/K) + (r + σ²/2)T] / (σ√T)
d2 = d1 - σ√T
```

**参数说明**：
- S = 标的当前价格
- K = 行权价
- T = 距到期时间（年）
- r = 无风险利率（默认5%）
- σ = 隐含波动率
- N(x) = 标准正态分布累积函数

### 盈亏计算

**单个持仓盈亏**：
```
P&L = (当前期权价值 - 开仓成本) × 数量 × 100

当前期权价值 = Black-Scholes计算的理论价格
开仓成本 = 开仓价格
数量 = 正数（买入）或负数（卖出）
```

**组合盈亏**：
```
总P&L = Σ(所有持仓的P&L)
```

## 常见问题

### Q1: 为什么卖出期权要用负数？
A: 负数表示卖出（Short）头寸。卖出期权时，你收取权利金，但承担义务。负数帮助系统正确计算盈亏方向。

### Q2: 隐含波动率如何选择？
A: 
- 查看期权市场的实际IV数据
- 历史波动率作为参考
- 典型范围：低波动15-20%，中等25-35%，高波动40%+
- 可以创建多个波动率情景进行对比

### Q3: 为什么我的盈亏和实际不一致？
A: 可能原因：
- 隐含波动率设置不准确
- 时间价值衰减（Theta）
- 实际市场价格与理论价格的差异
- 交易成本未计入

### Q4: 如何理解最大亏损？
A: 最大亏损是在设定价格范围内（±50%）可能出现的最大损失。实际亏损可能：
- 更大（如果价格超出范围）
- 更小（如果提前平仓）
- 受到时间价值衰减影响

### Q5: 系统支持哪些期权策略？
A: 系统支持任意组合策略，包括：
- 单腿：Long/Short Call/Put
- 价差：Bull/Bear Spread
- 跨式：Straddle, Strangle
- 蝶式：Butterfly
- 铁鹰：Iron Condor
- 自定义组合

## 最佳实践

1. **定期更新价格和波动率**
   - 市场变化时重新计算
   - 关注重大事件对IV的影响

2. **设置风险警戒线**
   - 注意最大亏损金额
   - 考虑账户承受能力

3. **多情景分析**
   - 测试不同波动率下的表现
   - 考虑极端市场情况

4. **时间管理**
   - 关注临近到期的合约
   - 考虑时间价值衰减

5. **组合优化**
   - 平衡风险和收益
   - 分散持仓
